AIDE : Gestionnaire des utilisateurs MX

Generalites

Cette application comporte 6 onglets.

  • Administration : se passe de commentaire.
  • Options : restaure les configurations par defaut de l'utilisateur choisi.
  • Copier/Synchroniser : copie ou cree un lien symbolique du dossier d'un utilisateur vers un autre. Si vous utilisez un navigateur autre qu'un navigateur Mozilla, vous devrez copier manuellement le dossier cache dans votre Home (par exemple, .opera).
  • Liberer de l'espace : propose plusieurs actions simples pour verifier et augmenter l'espace disque disponible.
  • Ajouter/Retirer des groupes : se passe de commentaire.
  • Appartenance a un groupe : permet a l'administrateur de modifier l'acces d'un utilisateur donne aux services et aux applications.

Notes

Onglet Options : Restaurer les valeurs par defaut : valeurs par defaut de Xfce
Lorsque l'utilisateur choisit de restaurer les parametres d'origine de Xfce dans MX Linux, le contenu de deux "fichiers squelettes" est copie dans le dossier personnel de l'utilisateur :

  • /etc/skel/.config/, vers ~/.config
  • /etc/skel/.gtk-2.0/, vers ~/.gtk-2.0/

Aucun autre fichier n'est remplace. Si l'utilisateur change plus tard d'avis, les fichiers remplaces se trouvent dans le dossier cache ~/.restore de son dossier personnel. Il suffit alors de recopier ces dossiers archives dans Home et d'accepter l'ecrasement.
