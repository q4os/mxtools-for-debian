MX Samba Config Help

MX Samba Config is the MX Linux tool for managing Samba users and user shares from a Qt interface.

Main Tasks

  • Enable or disable automatic Samba startup.
  • Start or stop the Samba service.
  • Add or remove Samba users.
  • Create, edit, and remove shared folders.

Users

Use the user controls to add Samba passwords for existing system users or remove Samba access when it is no longer needed.

Shares

The share list shows folders exported through Samba usershares. Add a new share for a folder you own, or edit an existing share to change its name, comment, guest access, write access, or visibility.

Service Controls

The Samba buttons at the bottom of the main window let you control whether the service starts automatically and whether it is currently running.

Notes

  • Shares typically need to point to folders owned by the user creating them.
  • Remote systems may need to reconnect after share or permission changes.
  • Package documentation, including the license and changelog, is installed under /usr/share/doc/mx-samba-config/.
