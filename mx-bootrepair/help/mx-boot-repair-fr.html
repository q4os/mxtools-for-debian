AIDE : Réparation d’amorçage MX

Procédure

  • Amorcer le LiveMedium et cliquer sur Menu démarrer ≻ Système ≻ MX Réparation d’amorçage, puis sélectionner l’option appropriée.
  • GRUB
    • Réinstaller un nouveau chargeur d’amorçage.
    • Réparer le chargeur d’amorçage en régénérant le fichier de configuration de GRUB (grub.cfg). C’est l’utilisation la plus courante de cette application, lorsqu’il est impossible de démarrer une partition depuis laquelle l’utilisateur pourrait lancer update-grub.
  • Bloc d’initialisation : ATTENTION, uniquement pour les systèmes démarrant en mode hérité (legacy) qui s’appuient sur le BIOS plutôt que sur l’UEFI.
    • Sauvegarder le bloc d’initialisation, soit le bloc maître (MBR), soit le bloc de partition (PBR).
    • Restaurer un bloc d’initialisation sauvegardé auparavant.
